# Typescript + NodeJS + Prisma + PostgreSQL


## Start Example
```
1. 👉 yarn install

2. 👉 bash start.sh

❗ WAIT, DON'T REQUEST ANYTHING YET ❗

3. 👉 yarn migrate

3. ✔️ READY TO RECEIVE REQUESTS!
```

## Prisma
```
# Initialize Prisma in project
1. 👉 yarn run prisma init 

# Push schema to database
2. 👉 yarn migrate

# Alter Prisma data
3. 👉 yarn run prisma studio
```
